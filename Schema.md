# Схема зависимостей между концепциями

## Структурная иерархия

Это диаграмма рекомендуемого порядка изучения тем. Стрелка → означает "нужно понимать прежде, чем переходить к".

### Уровень 1: Фундамент

```
Системы отсчета
      ↓
Вектор и операции над векторами
```

### Уровень 2: Кинематика точки (основы)

```
Системы отсчета + Векторы
      ↓
Траектория, путь, перемещение
      ↓
Скорость (вектор и её смысл)
      ↓
Ускорение (вектор и его смысл)
```

### Уровень 3: Координатные системы

```
Скорость и ускорение в декартовых
      ↓
┌──────────────┬───────────────────┐
↓              ↓                   ↓
Полярные     Естественные      Цилиндрические
координаты   координаты        координаты
└──────────────┼───────────────────┘
      ↓
Сферические координаты (опционально)
```

### Уровень 4: Приложения кинематики точки

```
Скорость и ускорение в разных координатах
      ↓
┌─────────────────┬──────────────────┐
↓                 ↓                  ↓
Круговое       Движение в      Проекции
движение       полярных        снаряда и т.п.
               координатах
```

### Уровень 5: Твёрдое тело

```
Ускорение, угловые величины
      ↓
Поступательное движение
      ↓
Вращение вокруг оси
      ↓
Плоское движение
      ↓
Мгновенный центр вращения (MCV)
      ↓
Движение в пространстве (опционально)
```

### Уровень 6: Сложное движение (высокий уровень)

```
Кинематика твёрдого тела
+ Преобразования систем отсчета
      ↓
Абсолютное, относительное, переносное
      ↓
Композиция скоростей
      ↓
Композиция ускорений
      ↓
Вращающиеся системы отсчета
```

## Рекомендуемые последовательности чтения

### Минимальный путь (6–8 часов)
1. Системы отсчета
2. Траектория, скорость, ускорение в Декартовых координатах
3. Круговое движение
4. Поступательное и вращательное движение твёрдого тела
5. Плоское движение и MCV

### Стандартный курс (12–15 часов)
1. Системы отсчета
2. Вектор и операции (краткий обзор)
3. Траектория, скорость, ускорение
4. Декартовы, полярные, естественные координаты
5. Примеры: круговое движение, снаряд
6. Твёрдое тело: поступательное, вращательное, плоское
7. Мгновенный центр вращения
8. Введение в сложное движение

### Расширенный курс (20+ часов)
Весь стандартный путь + :
- Цилиндрические и сферические координаты
- Сложное движение во всех деталях
- Преобразования между системами отсчета
- Вращающиеся системы отсчета и фиктивные силы

## Примеры перекрёстных ссылок

Многие темы пересекаются и могут быть связаны:

- **Круговое движение** ← используется в примерах для:
  - Полярных координат
  - Вращательного движения твёрдого тела
  - Естественных координат
  
- **Мгновенный центр вращения** ← может быть применён к:
  - Плоскому движению твёрдого тела
  - Некоторым случаям сложного движения
  
- **Композиция скоростей** ← нужна для:
  - Понимания сложного движения
  - Преобразований между системами отсчета

## Примечания по адаптивности

- **Для физиков**: можно сосредоточиться на Уровнях 1–4 и базовом 5
- **Для инженеров**: уделить больше внимания твёрдому телу (Уровень 5) и приложениям
- **Для математиков**: расширить разделы о координатных системах и преобразованиях
