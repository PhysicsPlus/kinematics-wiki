# Навигатор по документации проекта

## Основные файлы (источник правды)

| Файл | Что содержит |
|------|-------------|
| `CLAUDE.md` | Правила оформления заметок (авточитаемый) |
| `context/decisions.md` | Архитектура, курс, источники, конвертация Md→LaTeX |
| `context/current-work.md` | Статус работы, промпты для чатов |
| `context/briefing.md` | Самодостаточная инструкция для нейросетей |
| `conventions/notation.md` | Реестр обозначений, рисунки |

## Файлы для агентов Claude Code

| Файл | Что содержит |
|------|-------------|
| `.claude/memory/rules_cached.md` | Сжатые правила (чеклист без пояснений) |
| `.claude/memory/workflows/math.md` | Промпт: проверка раздела Математика |
| `.claude/memory/workflows/physics.md` | Промпт: проверка разделов физики |

## Быстрый старт нового сеанса

1. `CLAUDE.md` читается автоматически
2. Для написания заметок → промпт из `context/current-work.md`
3. Для проверки оформления → workflow из `.claude/memory/workflows/`
