# Правила оформления (сжатая версия)

Полная версия: `CLAUDE.md` в корне проекта. Здесь — только чеклист для быстрой проверки.

## LaTeX

- `\frac{}{}` везде; косая дробь только в показателе степени: `e^{-t^2/2}`, `x^{1/2}`
- `\int\limits_a^b`, не `\int_a^b`
- Не использовать `\dfrac`, `\displaystyle`, `\boxed{}`
- Тригонометрия: `\operatorname{tg}`, `\operatorname{ctg}`, `\operatorname{arctg}`, `\operatorname{arcctg}`
- Векторы латинские: `\mathbf{v}`; греческие: `\boldsymbol{\Omega}`; орты: `\hat{\mathbf{e}}_x`
- Не использовать `\vec{}`
- Производные по времени: `\dot{x}`, `\ddot{x}`
- Дифференциал: `\mathrm{d}t`
- Числа: `0{,}5` (запятая), единицы: `5~\text{м/с}`
- Перенос формул: `aligned`, строка кончается на `=`, следующая начинается с `=`
- Несколько формул: `gathered` (по центру) или `aligned` с `&` (по левому краю)
- `$$...$$` без пустых строк вокруг

## Структура заметки

- Аннотация → `## ☆/★/⭐` секции → `## Связи и продолжения`
- Заголовки только `##`, запрещено `###`
- Нет `---` разделителей, нет `## LaTeX-код`
- Жирный — только для нового термина при первом вводе

## Callout-блоки

- `[!definition]` — точные определения и ключевые законы
- `[!example]` — inline-примеры (заголовок + условие + решение; без жирного в теле)
- Длинные примеры → отдельный файл `examples/*/пример-*.md`
