# Пример: кривошипно-шатунный механизм

## Условие

Кривошип $OA$ длины $a = 0{,}2~\text{м}$ вращается вокруг неподвижной точки $O$ с угловой скоростью $\omega_0 = 5~\text{рад/с}$. Шатун $AB$ длины $b = 0{,}5~\text{м}$ соединяет точку $A$ с ползуном $B$, который может двигаться только горизонтально.

В рассматриваемый момент кривошип $OA$ горизонтален и направлен вправо, шатун $AB$ тоже горизонтален. Найдите скорость ползуна $v_B$ и угловую скорость шатуна $\omega_{AB}$.

## Решение через теорему о проекциях

Скорость точки $A$: $v_A = \omega_0 a = 5\cdot0{,}2 = 1~\text{м/с}$, направлена вертикально вверх (перпендикулярно горизонтальному кривошипу).

Применим теорему о проекциях к шатуну $AB$ (горизонтальному). Проекция $\mathbf{v}_A$ на горизонталь равна нулю. Следовательно, проекция $\mathbf{v}_B$ на горизонталь тоже равна нулю. Но $\mathbf{v}_B$ сам горизонтален — значит:
$$v_B = 0$$

В данный момент ползун мгновенно покоится — это **мёртвая точка** механизма.

## Нахождение $\omega_{AB}$

Применим формулу поля скоростей к шатуну $AB$. Ось $y$ — вертикально вверх. Координаты: $v_{Ay} = 1~\text{м/с}$, $v_{By} = 0$, $x_{AB} = b = 0{,}5~\text{м}$.

Из формулы $v_{By} = v_{Ay} + \omega_{AB}\cdot x_{AB}$:
$$0 = 1 + \omega_{AB}\cdot 0{,}5 \implies \omega_{AB} = -2~\text{рад/с}$$

Отрицательный знак: шатун вращается по часовой стрелке.

## Геометрическое объяснение мёртвой точки

В данной конфигурации скорость $A$ направлена вертикально, а шатун горизонтален — значит $\mathbf{v}_A \perp AB$. МЦС шатуна лежит на перпендикуляре к $\mathbf{v}_A$ через $A$ (это горизонталь через $A$) и на перпендикуляре к $\mathbf{v}_B$ через $B$ (это вертикаль через $B$). Но $A$ и $B$ расположены на одной горизонтали — МЦС шатуна совпадает с $B$. Скорость $B$ равна нулю — подтверждает мёртвую точку.
