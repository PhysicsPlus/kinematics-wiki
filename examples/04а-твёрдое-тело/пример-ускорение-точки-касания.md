# Пример: ускорение точки касания катящегося диска

## Условие

Диск радиуса $R$ катится без скольжения вправо. Скорость центра $O$ равна $v$, ускорение центра горизонтально и равно $a_O$. Найдите ускорение точки касания $C$ с опорой.

## Почему это интересно

Точка касания $C$ является мгновенным центром скоростей, поэтому $\mathbf{v}_C = \mathbf{0}$. Казалось бы, раз скорость равна нулю, то и ускорение должно быть нулевым? Этот пример опровергает такую логику.

## Решение

При качении без скольжения: $\omega = v/R$ (вращение по часовой стрелке, $\omega < 0$ в соглашении «против часовой — положительно»), $\varepsilon = -a_O/R$.

Выбираем полюс $A = O$ (центр диска), вычисляем ускорение точки $B = C$ (точки касания).

Вектор $\overrightarrow{OC}$ направлен вертикально вниз: $x_{OC} = 0$, $y_{OC} = -R$.

Подставляем в формулу Эйлера:
$$a_{Cx} = a_{Ox} - \varepsilon\, y_{OC} - \omega^2\, x_{OC}$$
$$= a_O - \left(-\frac{a_O}{R}\right)(-R) - \omega^2\cdot 0 = a_O - a_O = 0$$

$$a_{Cy} = a_{Oy} + \varepsilon\, x_{OC} - \omega^2\, y_{OC}$$
$$= 0 + \varepsilon\cdot 0 - \omega^2(-R) = \omega^2 R = \frac{v^2}{R}$$

## Ответ

$$\mathbf{a}_C = \left(0,\; \frac{v^2}{R}\right)$$

Ускорение точки касания направлено **вертикально вверх** (к центру диска) и равно $v^2/R$.

## Физический смысл

Точка касания мгновенно покоится, но немедленно начнёт двигаться в следующий момент — она «поднимается» вверх по диску. Ускорение $v^2/R$ — это именно центростремительное ускорение, необходимое для искривления траектории этой точки.

Это показывает: **нулевая скорость в данный момент не означает нулевого ускорения**. Ускорение МЦС ненулевое — именно потому метод МЦС неприменим к ускорениям. В формуле Эйлера центростремительное слагаемое $\omega^2|AB|$ не исчезает, даже если $B$ — МЦС.
