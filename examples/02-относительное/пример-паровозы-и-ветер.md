# Пример: паровозы и шлейф дыма

## Условие

По прямолинейному участку дороги движутся два паровоза со скоростями $v_1 = 50~\text{км/ч}$ и $v_2 = 70~\text{км/ч}$ (вид сверху). На фотографии видны шлейфы дыма от обоих паровозов (направления шлейфов указаны на рисунке). Определите скорость ветра.

## Ключевая идея

Дым выходит из трубы со скоростью, близкой к нулю в системе воздуха, и затем движется вместе с ветром. В системе отсчёта паровоза дым отлетает назад (в сторону, противоположную движению) плюс сносится ветром. Направление шлейфа в системе паровоза = $\vec{v}_{\text{ветер}} - \vec{v}_{\text{паровоза}}$ (скорость ветра относительно паровоза).

## Решение

Пусть скорость ветра относительно земли — $\vec{u}$. Скорость ветра относительно паровоза $i$:
$$\vec{u}_i = \vec{u} - \vec{v}_i$$

Шлейф дыма за паровозом направлен вдоль $\vec{u}_i$ (ветер «тянет» дым в эту сторону). Из рисунка известны направления $\vec{u}_1$ и $\vec{u}_2$.

Строим систему уравнений: по двум известным направлениям $\vec{u}_i = \vec{u} - \vec{v}_i$ находим $\vec{u}$. Геометрически: $\vec{u} = \vec{v}_i + \vec{u}_i$. Мы знаем $\vec{v}_i$ (горизонтально вдоль пути) и направление $\vec{u}_i$. Из каждого паровоза получаем луч в пространстве скоростей, на котором лежит $\vec{u}$. Пересечение двух лучей — и есть скорость ветра.

Численный ответ зависит от конкретных углов на рисунке (задача графическая); метод — одинаков.

## Приём

Общий метод: если дым движется в системе ветра, то направление шлейфа = скорость ветра в системе паровоза. По двум паровозам с известными скоростями строим два луча и находим $\vec{u}$ как их пересечение. Аналогично решается задача «по фотографии найти скорость ветра» для любого числа движущихся объектов с дымом.
