# Пример: паровозы и шлейф дыма

## Условие

По прямолинейному участку дороги движутся два паровоза со скоростями $v_1 = 50~\text{км/ч}$ и $v_2 = 70~\text{км/ч}$ (вид сверху). На фотографии видны шлейфы дыма от обоих паровозов (направления шлейфов указаны на рисунке). Определите скорость ветра.

## Ключевая идея

Дым выходит из трубы со скоростью, близкой к нулю в системе воздуха, и затем движется вместе с ветром. В системе отсчёта паровоза дым отлетает назад (в сторону, противоположную движению) плюс сносится ветром. Направление шлейфа в системе паровоза = $\mathbf{v}_{\text{ветер}} - \mathbf{v}_{\text{паровоза}}$ (скорость ветра относительно паровоза).

## Решение

Пусть скорость ветра относительно земли — $\mathbf{u}$. Скорость ветра относительно паровоза $i$:
$$\mathbf{u}_i = \mathbf{u} - \mathbf{v}_i$$

Шлейф дыма за паровозом направлен вдоль $\mathbf{u}_i$ (ветер «тянет» дым в эту сторону). Из рисунка известны направления $\mathbf{u}_1$ и $\mathbf{u}_2$.

Строим систему уравнений: по двум известным направлениям $\mathbf{u}_i = \mathbf{u} - \mathbf{v}_i$ находим $\mathbf{u}$. Геометрически: $\mathbf{u} = \mathbf{v}_i + \mathbf{u}_i$. Мы знаем $\mathbf{v}_i$ (горизонтально вдоль пути) и направление $\mathbf{u}_i$. Из каждого паровоза получаем луч в пространстве скоростей, на котором лежит $\mathbf{u}$. Пересечение двух лучей — и есть скорость ветра.

Численный ответ зависит от конкретных углов на рисунке (задача графическая); метод — одинаков.

## Приём

Общий метод: если дым движется в системе ветра, то направление шлейфа = скорость ветра в системе паровоза. По двум паровозам с известными скоростями строим два луча и находим $\mathbf{u}$ как их пересечение. Аналогично решается задача «по фотографии найти скорость ветра» для любого числа движущихся объектов с дымом.
