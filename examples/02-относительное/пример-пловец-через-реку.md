# Пример: пловец через реку

## Условие

Пловец хочет переплыть реку шириной $L$. Скорость течения равна $u$, скорость пловца относительно воды — $v$ ($v > u$).

1. Под каким углом к берегу нужно плыть, чтобы переправиться за наименьшее время? Какое расстояние он пройдёт?
2. За какое время пловец переплывёт реку по наикратчайшему пути?

## Ключевая идея

Абсолютная скорость пловца:
$$\mathbf{v}_\text{абс} = \mathbf{v}_\text{отн} + \mathbf{u}$$
где $\mathbf{u}$ — скорость течения (вдоль берега), $|\mathbf{v}_\text{отн}| = v$ — скорость относительно воды (направление выбирает пловец). Задача — выбрать направление $\mathbf{v}_\text{отн}$.

## Решение задачи 1: наименьшее время

Время переправы $t = L / v_\perp$, где $v_\perp$ — поперечная (к берегу) компонента абсолютной скорости. Поскольку $\mathbf{u}$ направлен вдоль берега, поперечная компонента $\mathbf{v}_\text{абс}$ совпадает с поперечной компонентой $\mathbf{v}_\text{отн}$.

Поперечная компонента $\mathbf{v}_\text{отн}$ максимальна, если пловец плывёт строго перпендикулярно берегу: $v_\perp^\max = v$. Время:
$$t_{\min} = \frac{L}{v}$$

При этом течение сносит пловца вдоль берега на $S_x = u \cdot t_{\min} = uL/v$. Пройденное расстояние:
$$S = \sqrt{L^2 + \left(\frac{uL}{v}\right)^2} = \frac{L}{v}\sqrt{u^2 + v^2}$$

## Решение задачи 2: кратчайший путь

Кратчайший путь — строго поперёк реки (расстояние $L$). Для этого абсолютная скорость $\mathbf{v}_\text{абс}$ должна быть направлена перпендикулярно берегу, то есть $\mathbf{v}_\text{абс} \perp \mathbf{u}$.

Из треугольника скоростей ($\mathbf{v}_\text{абс} \perp \mathbf{u}$, $|\mathbf{v}_\text{отн}| = v$, $|\mathbf{u}| = u$): пловец должен направить $\mathbf{v}_\text{отн}$ под углом $\varphi$ против течения, где $\sin\varphi = u/v$. Поперечная компонента:
$$v_\perp = v\cos\varphi = \sqrt{v^2 - u^2}$$

Время переправы по кратчайшему пути:
$$t = \frac{L}{\sqrt{v^2 - u^2}}$$

## Ответы

1. Плыть перпендикулярно берегу; $t_{\min} = L/v$; расстояние $S = \frac{L}{v}\sqrt{u^2+v^2}$.
2. $t = L/\sqrt{v^2 - u^2}$ (при условии $v > u$).
