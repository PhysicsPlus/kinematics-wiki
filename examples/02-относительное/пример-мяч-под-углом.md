# Пример: упругий удар о стенку под углом

## Условие

Упругий мячик летит в направлении движущейся от него стенки со скоростью $v$. Стенка движется от мяча со скоростью $u$, составляющей угол $\alpha = 30°$ с нормалью к стенке; скорость мяча вдвое больше скорости стенки: $v = 2u$. Определите скорость мяча после удара.

## Решение

Введём систему координат: ось $n$ перпендикулярна стенке (от стенки к мячу), ось $\tau$ — вдоль стенки.

Скорость стенки по компонентам:
$$u_n = u\cos30° = \frac{u\sqrt{3}}{2} \quad \text{(от мяча)}, \qquad u_\tau = u\sin30° = \frac{u}{2} \quad \text{(вдоль стенки)}$$

Скорость мяча в лабораторной системе направлена туда же, куда движется стенка (мяч летит к стенке):
$$(v_n,\; v_\tau) = (-v\cos30°,\; -v\sin30°) = (-u\sqrt{3},\; -u)$$
(знак минус у $v_n$: мяч летит к стенке, то есть против оси $n$).

Шаг 1. Скорость мяча в системе стенки:
$$(v'_n,\; v'_\tau) = (v_n - u_n,\; v_\tau - u_\tau) = \left(-u\sqrt{3} - \frac{u\sqrt{3}}{2},\; -u - \frac{u}{2}\right) = \left(-\frac{3u\sqrt{3}}{2},\; -\frac{3u}{2}\right)$$

Шаг 2. Упругое отражение в системе стенки: нормальная компонента меняет знак, тангенциальная сохраняется:
$$v''_n = +\frac{3u\sqrt{3}}{2}, \qquad v''_\tau = -\frac{3u}{2}$$

Шаг 3. Возврат в лабораторную систему:
$$v_{\text{после},n} = v''_n + u_n = \frac{3u\sqrt{3}}{2} + \frac{u\sqrt{3}}{2} = 2u\sqrt{3}$$
$$v_{\text{после},\tau} = v''_\tau + u_\tau = -\frac{3u}{2} + \frac{u}{2} = -u$$

## Ответ

Итоговая скорость $\vec{v}_\text{после} = (2u\sqrt{3},\; -u)$, модуль:
$$|\vec{v}_\text{после}| = \sqrt{(2u\sqrt{3})^2 + u^2} = \sqrt{12u^2 + u^2} = u\sqrt{13}$$
