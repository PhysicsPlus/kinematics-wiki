# Пример: ракета и пусковая установка

## Условие

Ракета удаляется от поверхности Земли со скоростью $\mathbf{v}$, направленной строго вертикально. Как должна двигаться (относительно Земли) система отсчёта, в которой ракета и пусковая установка имеют одинаковые по величине скорости, равные $v$?

## Решение

Из закона сложения скоростей $\mathbf{v}_{\text{абс}} = \mathbf{v}_{\text{пер}} + \mathbf{v}_{\text{отн}}$:
- $\mathbf{v}_{\text{абс}} = \mathbf{v}$ — абсолютная скорость ракеты (вертикально вверх, модуль $v$)
- $\mathbf{v}_{\text{пер}} = \mathbf{u}$ — искомая переносная скорость системы отсчёта
- $\mathbf{v}_{\text{отн}}$ — скорость ракеты в подвижной системе, по условию $|\mathbf{v}_{\text{отн}}| = v$

По условию скорость пусковой установки в подвижной системе тоже равна $v$ по модулю. Скорость пусковой установки в Земле равна нулю, поэтому её скорость в подвижной системе равна $-\mathbf{u}$, то есть $|\mathbf{u}| = v$.

Нужно разложить вектор $\mathbf{v}$ в сумму двух векторов $\mathbf{u}$ и $\mathbf{v}_{\text{отн}}$, каждый из которых имеет модуль $v$. Три вектора образуют равносторонний треугольник (все стороны равны $v$).

Из геометрии равностороннего треугольника угол при основании равен $60°$. Переносная скорость $\mathbf{u}$ составляет угол $\alpha = 30°$ с вертикалью, то есть $30°$ к горизонту.

## Ответ

Система отсчёта должна двигаться со скоростью $u = v$ под углом $\alpha = 30°$ к горизонту.
