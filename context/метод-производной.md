## Концептуальный вопрос: как вводить производную?

Производная помечена ⭐ (уровень 2–3) — потому что для уровня 1 (школьник 9 кл.)
достаточно понятия предела и мгновенной скорости. Производная как
формальный математический объект — уровень 2–3.

### Три подхода, которые нужно объединить в заметке:

**1. Физический подход (Яковлев, der.pdf; Мякишев §1.7)**
- Мгновенная скорость как предел $\Delta x/\Delta t$
- Ускорение как $dv/dt$
- Наклон касательной к графику $x(t)$ → скорость
- Площадь под $v(t)$ → перемещение
- Это даёт физическую мотивацию: зачем нужна производная

**2. Геометрический подход (lec_3.tex — авторские разработки)**
- Касательная как линейное приближение функции вблизи точки $x_0$
- Угловой коэффициент касательной $k$ через предел разностного отношения
- Производные $x^2$, $1/x$, $\sin x$ — получены геометрически из lec_3
- Связь: $v(t)$ линейная → $S(t)$ квадратичная

**3. Аналитический подход (Kleppner §1.8–1.10)**
- Разложение $\Delta\vec{A}$ на параллельную и перпендикулярную части
- Свойство: если $|\vec{A}| = \text{const}$, то $\dot{\vec{A}} \perp \vec{A}$
- Производная вращающегося единичного вектора: $\dot{\hat{e}} \perp \hat{e}$, $|\dot{\hat{e}}| = \omega$
- Это ключевой факт для кинематики в полярных координатах

## Ключевые примеры для заметки

Из lec_3.tex (авторские разработки):
- $f(x) = x^2$: $k = 2x_0$ — вывод через предел разностного отношения
- $f(x) = 1/x$: $k = -1/x_0^2$
- $f(x) = \sin x$: $k = \cos x_0$ — через формулу сдвига $\sin(x_0+\Delta x)$
- Из $v(t) = at + v_0$ линейной → $S(t) = At^2 + Bt$ квадратичной

Из Яковлева (der.pdf):
- $v_x(t) = a + bt^2$ → найти $v$ и $a$ в конкретный момент
- Гармонические колебания: $x = 5\sin 2t$ → $v = 10\cos 2t$, $a = -20\sin 2t$

Из Kleppner (§1.10):
- Доказательство: $|\vec{A}| = \text{const} \Rightarrow \vec{A}\cdot\dot{\vec{A}} = 0$
- Равномерное движение по окружности: $\vec{a} = -\omega^2\vec{r}$ (центростремительное)

## Связь производной вектора с кинематикой

Это критически важный раздел — «мост» между математикой и кинематикой:

$$\frac{d\hat{e}_r}{dt} = \omega\hat{e}_\varphi, \qquad \frac{d\hat{e}_\varphi}{dt} = -\omega\hat{e}_r$$

Эти формулы — основа всей кинематики в полярных координатах (гл. 05).
Вывести здесь, чтобы в гл. 05 только использовать.

## Что НЕ включать в заметку `производная`

- Строгое определение Коши (уже есть в `предел`)
- Полный вывод формул полярных координат (это гл. 05)
- Теорему Лагранжа о среднем значении (не нужна для кинематики)
- Дифференциалы (отдельная тема, при необходимости)
