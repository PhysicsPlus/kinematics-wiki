# Относительное движение и системы отсчёта

Переход между системами отсчёта — одна из центральных идей курса.
Изучается сразу после одномерной кинематики.

## Заметки

- [[системы-отсчёта]] — определение, равноправность, зачем переходить
- [[сложение-скоростей]] — закон сложения при поступательном движении СО
- [[метод-подвижной-системы]] — минимальное расстояние, переправа через реку
- [[отражение-от-движущейся-поверхности]] — метод перехода в СО стенки
- [[эффект-доплера]] ⭐ — движущийся источник и наблюдатель, асимметрия
- [[вращающаяся-система-скорости]] ⭐ — вращающийся базис, переносная скорость
- [[вращающаяся-система-ускорения]] ⭐ — кориолисово ускорение, центростремительное ($\boldsymbol{\Omega} = \text{const}$)
- [[теорема-кориолиса]] ⭐ — общий случай: переменная $\boldsymbol{\Omega}$, ускорение Эйлера, силы инерции
