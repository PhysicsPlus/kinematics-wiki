# Кинематика АТТ в 3D ⭐

Обобщение плоского движения на пространство. Наиболее общий случай движения абсолютно твёрдого тела.

## Планируемые заметки

- [[угловая-скорость-3d]] — вектор $\vec\omega$; его инвариантность; сложение угловых скоростей
- [[формула-эйлера-3d]] — $\vec{v}_B = \vec{v}_A + \vec\omega \times \vec{r}_{AB}$; вывод
- [[теорема-шаля-3d]] — любое мгновенное движение АТТ = поступательное + вращение вокруг оси
- [[винтовое-движение]] — мгновенная ось вращения, шаг винта $p = \vec{v}\cdot\hat{\omega}/|\vec\omega|$
- [[углы-эйлера]] — прецессия, нутация, собственное вращение; матрица поворота через углы Эйлера
