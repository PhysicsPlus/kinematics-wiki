# Интеграл ⭐

## Определение

**Определённый интеграл** $\int\limits_a^b f(x)\,dx$ — площадь фигуры, ограниченной графиком $f(x)$, осью $x$ и вертикалями $x=a$, $x=b$ (с учётом знака: площадь ниже оси отрицательна).

**Неопределённый интеграл** $\int f(x)\,dx = F(x) + C$, где $F'(x) = f(x)$ — первообразная.

**Теорема Ньютона–Лейбница:**
$$\int\limits_a^b f(x)\,dx = F(b) - F(a)$$

## Физический смысл в кинематике

Площадь под графиком $v_x(t)$ на $[t_1, t_2]$ равна изменению координаты:
$$x(t_2) - x(t_1) = \int\limits_{t_1}^{t_2} v_x(t)\,dt$$

Площадь под $a_x(t)$ — изменению скорости:
$$v_x(t_2) - v_x(t_1) = \int\limits_{t_1}^{t_2} a_x(t)\,dt$$

## Таблица интегралов

| $f(x)$ | $\int f(x)\,dx$ |
|--------|----------------|
| $x^n\ (n\neq-1)$ | $\frac{x^{n+1}}{n+1} + C$ |
| $\frac{1}{x}$ | $\ln|x| + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $e^x$ | $e^x + C$ |

## Интегрирование вектора

Интеграл вектора берётся покомпонентно:
$$\int \mathbf{v}(t)\,dt = \hat{\mathbf{e}}_x\int v_x\,dt + \hat{\mathbf{e}}_y\int v_y\,dt + \hat{\mathbf{e}}_z\int v_z\,dt$$

Это используется при нахождении $\mathbf{r}(t)$ из $\mathbf{v}(t)$, и $\mathbf{v}(t)$ из $\mathbf{a}(t)$.
