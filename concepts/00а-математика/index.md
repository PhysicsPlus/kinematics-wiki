# Математика: основы

Всё необходимое для работы с кинематикой — с нуля. Предполагается только знание квадратного уравнения и элементарной алгебры.

## Заметки

### Анализ: цепочка малые величины → производная

Заметки связаны сквозной логикой: каждая опирается на предыдущую и готовит почву для следующей.

| Заметка | Уровень | Статус | Ключевая идея |
|---------|---------|--------|---------------|
| [[малые-величины]] | 1–2 | ✅ | Линейные и квадратичные приближения; когда НЕ применять |
| [[оценка-точности]] | 1–2 | ✅ | Абсолютная и относительная погрешность; критерий отбрасывания слагаемого |
| [[геометрия-малых-углов]] | 2–3 ⭐ | ✅ | Дуга ≈ хорда в линейном; «провисание» $R\alpha^2/2$ в квадратичном; горизонт; $d\hat{\mathbf{e}}/dt$ |
| [[ряды-и-разложения]] | 2–3 ⭐ | ✅ | Бином, деление в столбик, ряд Тейлора, остаточный член |
| [[предел]] | 1–3 | ✅ | Определение Коши; $\lim\sin x/x = 1$; бесконечно малые; непрерывность |
| [[производная]] | 2–3 ⭐ | ✅ | Точный коэффициент линеаризации $C$; геом. смысл; правила; $d\hat{\mathbf{e}}/dt = \boldsymbol{\omega}\times\hat{\mathbf{e}}$ |
| [[интеграл]] | 2–3 ⭐ | ⚠️ черновик | Площадь под графиком; формула Ньютона–Лейбница |
| [[распространение-погрешностей]] | 1–2 | ✅ | Правила для сумм, произведений, степеней; накопление ошибок |
