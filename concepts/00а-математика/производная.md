# Производная ⭐

## Определение

Производная функции $f(x)$ в точке $x$ — предел отношения приращения функции к приращению аргумента:
$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

**Физический смысл:** производная — мгновенная скорость изменения величины. Скорость $v_x = \dot{x}$ и ускорение $a_x = \dot{v}_x = \ddot{x}$ — производные по времени.

**Геометрический смысл:** производная равна тангенсу угла наклона касательной к графику $f(x)$ в данной точке.

## Обозначения

$$f'(x) = \frac{df}{dx} = \dot{f} \quad (\text{точка — производная по времени})$$

## Таблица производных

| $f(x)$ | $f'(x)$ |
|--------|---------|
| $x^n$ | $nx^{n-1}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $1/x$ |
| $\text{const}$ | $0$ |

## Правила дифференцирования

$$[f \pm g]' = f' \pm g', \qquad [cf]' = cf'$$
$$[fg]' = f'g + fg' \qquad \text{(правило Лейбница)}$$
$$[f(g(x))]' = f'(g(x))\cdot g'(x) \qquad \text{(правило цепочки)}$$

## Производная вектора

Производная вектора берётся покомпонентно:
$$\frac{d\mathbf{r}}{dt} = \dot{x}\,\hat{\mathbf{e}}_x + \dot{y}\,\hat{\mathbf{e}}_y + \dot{z}\,\hat{\mathbf{e}}_z$$

Производная вектора постоянного модуля перпендикулярна самому вектору:
$$|\mathbf{A}| = \text{const} \implies \mathbf{A} \cdot \dot{\mathbf{A}} = 0$$
Это ключевое свойство используется при выводе нормального ускорения и условия нерастяжимости твёрдого тела.
