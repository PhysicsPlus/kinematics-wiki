# Тригонометрия

## Определения через единичную окружность

Рассмотрим окружность радиуса $1$ с центром в начале координат — **единичную окружность**. Возьмём точку $P$ на этой окружности. Угол $\alpha$ — это угол между положительной полуосью $OX$ и радиусом $OP$, отсчитываемый против часовой стрелки.

Тогда по определению:
$$\cos\alpha = x_P, \qquad \sin\alpha = y_P$$

где $(x_P, y_P)$ — координаты точки $P$.

Из этого определения сразу следует **основное тождество** — оно просто выражает, что точка лежит на единичной окружности:
$$\cos^2\alpha + \sin^2\alpha = 1$$

**Тангенс** определяется как отношение:
$$\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$$

Геометрически: если провести касательную к единичной окружности в точке $(1, 0)$, то $\tan\alpha$ — это $y$-координата пересечения луча $OP$ с этой касательной.

**Котангенс:** $\cot\alpha = \dfrac{\cos\alpha}{\sin\alpha} = \dfrac{1}{\tan\alpha}$.

## Значения тригонометрических функций в особых точках

Эти значения нужно знать наизусть:

| $\alpha$ | $0°$ | $30°$ | $45°$ | $60°$ | $90°$ | $120°$ | $180°$ |
|----------|------|-------|-------|-------|-------|--------|--------|
| $\alpha$, рад | $0$ | $\pi/6$ | $\pi/4$ | $\pi/3$ | $\pi/2$ | $2\pi/3$ | $\pi$ |
| $\sin\alpha$ | $0$ | $1/2$ | $\frac{\sqrt{2}}{2}$ | $\frac{\sqrt{3}}{2}$ | $1$ | $\frac{\sqrt{3}}{2}$ | $0$ |
| $\cos\alpha$ | $1$ | $\frac{\sqrt{3}}{2}$ | $\frac{\sqrt{2}}{2}$ | $1/2$ | $0$ | $-1/2$ | $-1$ |
| $\tan\alpha$ | $0$ | $\frac{1}{\sqrt{3}}$ | $1$ | $\sqrt{3}$ | $\infty$ | $-\sqrt{3}$ | $0$ |

**Мнемоника для $\sin$:** «ноль, половина, корень двух пополам, корень трёх пополам, один» — для углов $0°, 30°, 45°, 60°, 90°$.

**Как получить $\sin 30° = 1/2$:** в равностороннем треугольнике со стороной $1$ высота делит основание пополам. Угол при вершине $60°$, при основании $30°$. Высота равна $\sqrt{3}/2$, а сторона, противолежащая углу $30°$ — это половина стороны, то есть $1/2$.

**Как получить $\sin 45° = \sqrt{2}/2$:** в прямоугольном равнобедренном треугольнике с катетами $1$ и $1$ гипотенуза $= \sqrt{2}$. Синус $45°$ = отношение катета к гипотенузе $= 1/\sqrt{2} = \sqrt{2}/2$.

## Радианы: почему они удобнее градусов

В физике углы всегда измеряются в **радианах**. Радиан — это угол, при котором длина дуги равна радиусу. Для угла $\alpha$ в радианах:
$$\text{длина дуги} = R\alpha$$

Перевод: $180° = \pi$ рад, поэтому $1° = \pi/180 \approx 0{,}01745$ рад.

**Почему радианы удобнее?** Во-первых, формула дуги $s = R\alpha$ не содержит посторонних коэффициентов. Во-вторых (и это главное), приближение $\sin\alpha \approx \alpha$ верно только в радианах. В-третьих, производная $(\sin x)' = \cos x$ верна только при измерении в радианах (при градусах появился бы коэффициент $\pi/180$).

## Симметрия и знаки

Из определения через координаты точки на окружности:

- В I четверти ($0 < \alpha < \pi/2$): $\sin > 0$, $\cos > 0$, $\tan > 0$
- Во II четверти ($\pi/2 < \alpha < \pi$): $\sin > 0$, $\cos < 0$, $\tan < 0$
- В III четверти ($\pi < \alpha < 3\pi/2$): $\sin < 0$, $\cos < 0$, $\tan > 0$
- В IV четверти ($3\pi/2 < \alpha < 2\pi$): $\sin < 0$, $\cos > 0$, $\tan < 0$

**Мнемоника** «Все Студенты Тригонометрию Сдают»: по четвертям (I, II, III, IV) — Все (все положительные), Синус, Тангенс, Косинус.

**Чётность и нечётность:**
$$\sin(-\alpha) = -\sin\alpha \quad (\text{нечётная}), \qquad \cos(-\alpha) = \cos\alpha \quad (\text{чётная})$$

## Сдвиги на $\pi/2$ и $\pi$

Эти соотношения используются постоянно:

$$\sin\left(\frac{\pi}{2} - \alpha\right) = \cos\alpha, \qquad \cos\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha$$
$$\sin\left(\frac{\pi}{2} + \alpha\right) = \cos\alpha, \qquad \cos\left(\frac{\pi}{2} + \alpha\right) = -\sin\alpha$$
$$\sin(\pi - \alpha) = \sin\alpha, \qquad \cos(\pi - \alpha) = -\cos\alpha$$
$$\sin(\pi + \alpha) = -\sin\alpha, \qquad \cos(\pi + \alpha) = -\cos\alpha$$

**Как запомнить:** при сдвиге на $\pi/2$ (нечётный кратный) синус и косинус **меняются местами**; знак определяется по исходной функции. При сдвиге на $\pi$ (чётный кратный) функции **не меняются местами**, но знак определяется по четверти.

## Формулы сложения

Это фундаментальные формулы, из которых выводится всё остальное:

$$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$
$$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$
$$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$
$$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$

**Вывод формулы $\cos(\alpha+\beta)$** через единичную окружность (один из способов). Рассмотрим два единичных вектора: $\vec{u} = (\cos\alpha, \sin\alpha)$ и $\vec{v} = (\cos\beta, \sin\beta)$. Их скалярное произведение:
$$\vec{u}\cdot\vec{v} = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$

С другой стороны, угол между ними равен $\alpha - \beta$, поэтому $\vec{u}\cdot\vec{v} = \cos(\alpha-\beta)$. Получаем:
$$\cos(\alpha-\beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$

Заменив $\beta \to -\beta$ (и используя чётность косинуса): $\cos(\alpha+\beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$ ✓.

**Проверка** формулы сложения. При $\alpha = \beta = \pi/4$:
$$\sin\!\left(\frac{\pi}{4}+\frac{\pi}{4}\right) = \sin\frac{\pi}{2} = 1$$
По формуле: $\sin\frac{\pi}{4}\cos\frac{\pi}{4} + \cos\frac{\pi}{4}\sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}\cdot\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}\cdot\frac{\sqrt{2}}{2} = \frac{1}{2}+\frac{1}{2} = 1$ ✓.

## Формулы двойного угла

Подставляем $\beta = \alpha$ в формулы сложения:

$$\sin 2\alpha = 2\sin\alpha\cos\alpha$$
$$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 1 - 2\sin^2\alpha = 2\cos^2\alpha - 1$$

**Полезные следствия** (формулы понижения степени):
$$\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}, \qquad \cos^2\alpha = \frac{1 + \cos 2\alpha}{2}$$

**Применение в кинематике.** Дальность броска под углом $\alpha$ к горизонту:
$$L = \frac{v_0^2\sin 2\alpha}{g}$$
Максимум при $2\alpha = 90°$, то есть при $\alpha = 45°$.

## Формулы половинного угла

Из формул понижения степени при замене $\alpha \to \alpha/2$:
$$\sin\frac{\alpha}{2} = \sqrt{\frac{1-\cos\alpha}{2}}, \qquad \cos\frac{\alpha}{2} = \sqrt{\frac{1+\cos\alpha}{2}}$$

**Применение.** Расстояние между двумя точками на окружности радиуса $R$, разделёнными углом $\alpha$:
$$|AB| = 2R\sin\frac{\alpha}{2}$$
Это длина хорды. При малом $\alpha$: $|AB| \approx 2R\cdot\frac{\alpha}{2} = R\alpha$ — длина дуги ✓.

## Формулы суммы и разности

$$\sin\alpha + \sin\beta = 2\sin\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$
$$\sin\alpha - \sin\beta = 2\cos\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$$
$$\cos\alpha + \cos\beta = 2\cos\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$
$$\cos\alpha - \cos\beta = -2\sin\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$$

**Вывод первой формулы.** Пусть $\alpha = x+y$, $\beta = x-y$ (обратная замена: $x = (\alpha+\beta)/2$, $y = (\alpha-\beta)/2$). Тогда:
$$\sin\alpha + \sin\beta = \sin(x+y) + \sin(x-y)$$
$$= (\sin x\cos y + \cos x\sin y) + (\sin x\cos y - \cos x\sin y) = 2\sin x\cos y$$
$$= 2\sin\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$

## Произведения тригонометрических функций

$$\sin\alpha\cos\beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]$$
$$\cos\alpha\cos\beta = \frac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]$$
$$\sin\alpha\sin\beta = \frac{1}{2}[\cos(\alpha-\beta) - \cos(\alpha+\beta)]$$

**Вывод:** непосредственно из формул сложения. Например:
$$\sin(\alpha+\beta) + \sin(\alpha-\beta) = 2\sin\alpha\cos\beta$$

## Проекции вектора

Вектор $\vec{a}$ модуля $|\vec{a}| = a$ направлен под углом $\varphi$ к оси $OX$. Его проекции:
$$a_x = a\cos\varphi, \qquad a_y = a\sin\varphi$$

И обратно: если известны проекции, то
$$a = \sqrt{a_x^2 + a_y^2}, \qquad \tan\varphi = \frac{a_y}{a_x}$$

Угол $\varphi$ определяется по знакам $a_x$ и $a_y$ (надо знать, в какой четверти лежит вектор).

**Важный частный случай.** Проекция вектора $\vec{a}$ на направление $\hat{n}$:
$$a_n = a\cos\theta$$
где $\theta$ — угол между $\vec{a}$ и $\hat{n}$. Это скалярное произведение $\vec{a}\cdot\hat{n}$.

## Тождества, полезные в задачах

$$1 + \tan^2\alpha = \frac{1}{\cos^2\alpha} \quad (\text{следует из } \cos^2 + \sin^2 = 1)$$
$$1 + \cot^2\alpha = \frac{1}{\sin^2\alpha}$$

**Теорема косинусов** (обобщение теоремы Пифагора):
$$c^2 = a^2 + b^2 - 2ab\cos C$$

где $C$ — угол между сторонами $a$ и $b$. При $C = 90°$ получаем теорему Пифагора.

**Теорема синусов:**
$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

где $R$ — радиус описанной окружности.
