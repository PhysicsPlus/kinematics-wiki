# Математика: Обыкновенные дифференциальные уравнения

Минимальный аппарат ОДУ, необходимый для глав по баллистике с сопротивлением и движению в центральном поле.

## Планируемые заметки

- [[переменные-разделяются]] — метод разделения переменных; примеры: $\dot{x} = f(x)g(t)$
- [[линейное-оду-1]] — линейное ОДУ первого порядка; метод вариации постоянной
- [[линейное-оду-2]] — ОДУ второго порядка с постоянными коэффициентами; характеристическое уравнение
- [[осциллятор]] — уравнение $\ddot{x} + \omega^2 x = 0$; решение, физический смысл
- [[численные-методы]] — метод Эйлера, качественная картина фазового портрета (кратко)

## Где используется

| Глава | Где нужно ОДУ |
|-------|---------------|
| [[concepts/06-баллистика/index\|06 Баллистика]] | сопротивление воздуха $\dot{v} = -kv^n$ |
| [[concepts/10-центральное-поле/index\|10 Центральное поле]] | уравнение Бине $u'' + u = f(u)$ |
| [[concepts/12-гироскопические/index\|12 Гироскопические силы]] | уравнение движения в магнитном поле |
