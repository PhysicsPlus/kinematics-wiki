# Тригонометрические формулы

## Суть

Набор формул, которые используются при выводах в кинематике: разложение вектора по компонентам, формулы двойного угла при выводе ускорений, формулы сложения при описании вращения.

## Основные соотношения

$$\sin^2\alpha + \cos^2\alpha = 1, \qquad \tan\alpha = \frac{\sin\alpha}{\cos\alpha}$$

## Формулы сложения

$$\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$$
$$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$$

## Двойной угол

$$\sin 2\alpha = 2\sin\alpha\cos\alpha$$
$$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 1 - 2\sin^2\alpha = 2\cos^2\alpha - 1$$

## Половинный угол

$$\sin^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}, \qquad \cos^2\frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}$$
$$1 + \cos\alpha = 2\cos^2\frac{\alpha}{2}, \qquad \sin\alpha = 2\sin\frac{\alpha}{2}\cos\frac{\alpha}{2}$$

(Формулы половинного угла используются при выводе угловой скорости вписанного угла в задаче о движении по окружности.)

## Проекции вектора

Вектор модуля $a$ под углом $\alpha$ к оси $OX$:
$$a_x = a\cos\alpha, \qquad a_y = a\sin\alpha$$
