# Равноускоренное движение

## Суть

Движение с постоянным ускорением $a_x = \text{const}$. Скорость изменяется линейно, координата — квадратично. Типичный пример — свободное падение (при пренебрежении сопротивлением).

## Формулы

Скорость:
$$v_x(t) = v_x(0) + a_x t$$

Координата:
$$x(t) = x(0) + v_x(0)\,t + \frac{a_x t^2}{2}$$

Связь скорости и перемещения (без времени):
$$x(t) - x(0) = \frac{v_x^2(t) - v_x^2(0)}{2a_x}$$

## Теория

**Графики:**
- $v_x(t)$ — прямая линия с наклоном $a_x$
- $x(t)$ — парабола

**Площадь под $v_x(t)$** на $[t_1, t_2]$ — это изменение координаты. Для знакопеременной скорости: площадь выше оси даёт перемещение в $+$, ниже — в $-$. Путь = сумма модулей площадей.

**Свободное падение** — частный случай с $a_x = -g$ (при оси $y$ вверх) или $a_x = +g$ (ось вниз). Условие применимости: пренебречь сопротивлением воздуха.

## LaTeX-код

```tex
При движении с постоянным ускорением:
\[
v_{x}(t) = v_{x}(0) + a_{x}t,
\]
\[
x(t) = x(0) + v_{x}(0)t +\frac{a_{x}t^{2}}{2},
\]
\[
x(t) - x(0) = \frac{ v_{x}^{2}(t)- v_{x}^{2}(0)}{2a_{x}}.
\]
```

## Замечания

- Формула без времени удобна, когда время не нужно — находить сразу $v$ через перемещение.
- Свободное падение: $g = 10\ \text{м/с}^2$ (приближение для задач).
