# Равноускоренное движение

## Зачем это нужно

Из бесконечного разнообразия движений с переменной скоростью есть одно особенно важное: движение с **постоянным ускорением** ($a_x = \text{const}$). Оно встречается в природе повсеместно:
- свободное падение (вблизи Земли, без сопротивления воздуха)
- разгон или торможение автомобиля с постоянным усилием
- движение в постоянном электрическом поле

Для этого случая формулы кинематики выводятся аналитически и дают три компактных уравнения, которые охватывают всю задачу целиком.

## Вывод формулы для скорости

Если ускорение постоянно: $a_x(t) = a_x = \text{const}$, то из определения ускорения $a_x = \dot{v}_x$ следует обыкновенное дифференциальное уравнение:
$$\frac{\mathrm{d}v_x}{\mathrm{d}t} = a_x$$

**Геометрический вывод.** На графике $v_x(t)$ ускорение — наклон прямой. Если $a_x = \text{const}$, то $v_x(t)$ — прямая. Прямая, проходящая через точку $(0, v_0)$ с наклоном $a_x$:
$$v_x(t) = v_0 + a_x t$$
где $v_0 = v_x(0)$ — начальная скорость.

**Аналитический вывод.** Разделяем переменные и интегрируем от $0$ до $t$:
$$\int\limits_0^t \mathrm{d}v_x = \int\limits_0^t a_x\,\mathrm{d}\tau$$
$$v_x(t) - v_0 = a_x t \implies v_x(t) = v_0 + a_x t$$

Скорость растёт линейно по времени.

---

**Пример 1. Разгон мотоцикла.**

Мотоцикл стартует из покоя ($v_0 = 0$) с ускорением $a_x = 3~\text{м/с}^2$.
$$v_x(t) = 0 + 3t = 3t~\text{м/с}$$
Через $t = 5~\text{с}$: $v_x = 15~\text{м/с} = 54~\text{км/ч}$.

---

## Вывод формулы для координаты

Из определения скорости $v_x = \dot{x}$ и только что полученного $v_x(t) = v_0 + a_x t$:

**Геометрический вывод.** Площадь под графиком $v_x(t) = v_0 + a_x t$ на промежутке $[0, t]$ — трапеция с основаниями $v_0$ и $v_0 + a_x t$ и высотой $t$:
$$\Delta x = x(t) - x_0 = \frac{(v_0) + (v_0 + a_x t)}{2}\cdot t = v_0 t + \frac{a_x t^2}{2}$$

**Площадь под $v_x(t)$ — это изменение координаты.** Это утверждение верно всегда (не только при $a_x = \text{const}$), но в общем случае требует понятия интеграла. Здесь — частный случай, геометрически прозрачный.

**Аналитический вывод.** Интегрируем $v_x(t) = v_0 + a_x t$ по времени:
$$x(t) - x_0 = \int\limits_0^t (v_0 + a_x\tau)\,\mathrm{d}\tau = v_0 t + \frac{a_x t^2}{2}$$
Обозначим начальную координату $x_0 = x(0)$:
$$x(t) = x_0 + v_0 t + \frac{a_x t^2}{2}$$

График $x(t)$ — парабола (при $a_x \neq 0$): ветви вверх при $a_x > 0$, вниз при $a_x < 0$.

---

**Пример 2. Мяч, брошенный вертикально вверх.**

Ось $y$ направлена вверх, начальная высота $y_0 = 0$, начальная скорость $v_0 = 20~\text{м/с}$, ускорение $a_y = -g = -10~\text{м/с}^2$.
$$y(t) = 20t - 5t^2, \quad v_y(t) = 20 - 10t$$
Момент остановки (верхняя точка): $v_y = 0 \Rightarrow t_* = 2~\text{с}$.

Максимальная высота: $y(2) = 20\cdot 2 - 5\cdot 4 = 40 - 20 = 20~\text{м}$.

Возвращение: $y(t) = 0 \Rightarrow t(20 - 5t) = 0 \Rightarrow t = 0$ (старт) или $t = 4~\text{с}$ (возврат).

Путь: $L = 20 + 20 = 40~\text{м}$. Перемещение: $\Delta y = 0$.

---

## Формула без времени

Из двух уравнений $v_x = v_0 + a_x t$ и $x - x_0 = v_0 t + \frac{a_x t^2}{2}$ можно исключить $t$.

Из первого: $t = \frac{v_x - v_0}{a_x}$. Подставляем во второе:
$$x - x_0 = v_0 \cdot \frac{v_x - v_0}{a_x} + \frac{a_x}{2}\cdot\frac{(v_x - v_0)^2}{a_x^2} = \frac{v_0(v_x - v_0)}{a_x} + \frac{(v_x-v_0)^2}{2a_x}$$
$$= \frac{2v_0(v_x-v_0) + (v_x-v_0)^2}{2a_x} = \frac{(v_x-v_0)(2v_0 + v_x - v_0)}{2a_x} = \frac{(v_x-v_0)(v_x+v_0)}{2a_x} = \frac{v_x^2 - v_0^2}{2a_x}$$

Итого:
$$v_x^2 = v_0^2 + 2a_x(x - x_0)$$

Эта формула связывает скорость и перемещение напрямую, без времени. Удобна, когда время — не нужная или неизвестная величина.

---

**Пример 3. Тормозной путь.**

Автомобиль едет со скоростью $v_0 = 108~\text{км/ч} = 30~\text{м/с}$. Тормозное ускорение $a_x = -6~\text{м/с}^2$. Какой путь пройдёт до остановки?

При остановке $v_x = 0$. Из формулы без времени:
$$0 = (30)^2 + 2(-6)\Delta x \implies \Delta x = \frac{900}{12} = 75~\text{м}$$

При скорости 108 км/ч тормозной путь — 75 м. Это 15 длин автомобиля.

## Сводная таблица уравнений

| № | Формула | Связывает |
|---|---------|-----------|
| 1 | $v_x = v_0 + a_x t$ | $v$, $a$, $t$ |
| 2 | $x = x_0 + v_0 t + \frac{a_x t^2}{2}$ | $x$, $v_0$, $a$, $t$ |
| 3 | $v_x^2 = v_0^2 + 2a_x(x - x_0)$ | $v$, $v_0$, $a$, $x$ |

Из трёх уравнений использовать нужное — то, из которого можно найти искомую величину по данным. Всегда проверять: $v_x = 0$ — это остановка, а не только конечная точка.

## Графики и геометрический смысл

**График $v_x(t)$** — прямая линия с наклоном $a_x$ и начальным значением $v_0$.

**График $x(t)$** — парабола. Если $a_x > 0$: ветви вверх; если $a_x < 0$: ветви вниз. Вершина параболы соответствует моменту остановки ($v_x = 0$).

**Площадь под $v_x(t)$ = изменение координаты.** Это фундаментальное соотношение, справедливое при любом законе движения:
$$x(t_2) - x(t_1) = \int\limits_{t_1}^{t_2} v_x(t)\,\mathrm{d}t$$
(подробнее — [[интеграл]]). При $a_x = \text{const}$ площадь — трапеция, что даёт формулу 2 выше.

**Площадь со знаком.** Если $v_x < 0$ на части промежутка, то соответствующая часть площади отрицательна и вычитается из перемещения. Путь равен сумме **модулей** площадей на всех участках знакопостоянства $v_x$.

---

**Пример 4. Перемещение при смене знака скорости.**

Тело движется с $v_0 = 6~\text{м/с}$, $a_x = -2~\text{м/с}^2$. Найти путь и перемещение за $t = 5~\text{с}$.

Момент остановки: $v_x(t_*) = 0 \Rightarrow t_* = 6 - 2t_* = 0 \Rightarrow t_* = 3~\text{с}$.

(Проверим: $t_* = 3~\text{с}$ лежит внутри $[0, 5~\text{с}]$ — тело действительно разворачивается.)

Перемещение:
$$\Delta x = 6\cdot 5 + \frac{(-2)\cdot 25}{2} = 30 - 25 = +5~\text{м}$$

Путь — складываем участки. До $t_* = 3~\text{с}$:
$$L_1 = 6\cdot 3 + \frac{(-2)\cdot 9}{2} = 18 - 9 = 9~\text{м}$$
После $t_* = 3~\text{с}$ (начальная скорость 0, ускорение $-2~\text{м/с}^2$, время $5-3=2~\text{с}$):
$$L_2 = 0 + \frac{(-2)\cdot 4}{2} = -4~\text{м} \Rightarrow |L_2| = 4~\text{м}$$

Путь: $L = 9 + 4 = 13~\text{м}$. Перемещение: $\Delta x = +5~\text{м}$.

Проверка: $9 - 4 = +5~\text{м}$ — совпадает. ✓

## Свободное падение

Частный случай равноускоренного движения: ускорение $g \approx 9{,}8~\text{м/с}^2$ вниз.

Выбор знака зависит от направления оси:
- Ось $y$ **вверх**: $a_y = -g$
- Ось $y$ **вниз**: $a_y = +g$

В задачах удобно округлять $g \approx 10~\text{м/с}^2$, если не оговорено иное.

---

**Пример 5. Бросок тела с некоторой высоты.**

Камень бросают вертикально вниз с крыши здания высотой $H = 45~\text{м}$ с начальной скоростью $v_0 = 5~\text{м/с}$ (ось $y$ вниз, $y_0 = 0$). Когда камень ударится о землю?

$y(t) = 5t + 5t^2 = 45 \Rightarrow 5t^2 + 5t - 45 = 0 \Rightarrow t^2 + t - 9 = 0$

$$t = \frac{-1 + \sqrt{1 + 36}}{2} = \frac{-1 + \sqrt{37}}{2} \approx \frac{-1 + 6{,}08}{2} \approx 2{,}54~\text{с}$$

(Отрицательный корень $t < 0$ не имеет физического смысла.)

---

**Пример 6. Два тела с разными стартами (задача о полицейском).**

Нарушитель едет с постоянной скоростью $v_1 = 30~\text{м/с}$. В момент $t = 0$ мимо стоящего полицейского проезжает нарушитель. Полицейский начинает разгон с ускорением $a = 5~\text{м/с}^2$.

Координаты ($x_0 = 0$ для обоих):
$$x_{\text{нар}}(t) = 30t, \qquad x_{\text{пол}}(t) = \frac{5t^2}{2}$$
Когда полицейский догоняет нарушителя: $\frac{5t^2}{2} = 30t \Rightarrow t(5t - 60) = 0 \Rightarrow t = 0$ или $t = 12~\text{с}$.

Через $t = 12~\text{с}$. Скорость полицейского: $v = 5\cdot 12 = 60~\text{м/с}$ — вдвое больше, чем у нарушителя.

Расстояние: $x = 30\cdot 12 = 360~\text{м}$.

## ⭐ Строгий вывод через интегрирование уравнения движения

Исходное уравнение: $\ddot{x} = a = \text{const}$. Это обыкновенное дифференциальное уравнение второго порядка.

**Первое интегрирование:** $\frac{\mathrm{d}v_x}{\mathrm{d}t} = a$. Интегрируем обе части по $t$ от $0$ до $t$:
$$\int\limits_0^t \frac{\mathrm{d}v_x}{\mathrm{d}\tau}\,\mathrm{d}\tau = \int\limits_0^t a\,\mathrm{d}\tau$$
$$\bigl[v_x(\tau)\bigr]_0^t = a\bigl[\tau\bigr]_0^t$$
$$v_x(t) - v_x(0) = at \implies v_x(t) = v_0 + at$$

**Второе интегрирование:** $\frac{\mathrm{d}x}{\mathrm{d}t} = v_0 + at$. Интегрируем:
$$\int\limits_0^t \frac{\mathrm{d}x}{\mathrm{d}\tau}\,\mathrm{d}\tau = \int\limits_0^t (v_0 + a\tau)\,\mathrm{d}\tau$$
$$x(t) - x(0) = v_0 t + \frac{at^2}{2} \implies x(t) = x_0 + v_0 t + \frac{at^2}{2}$$

Общее решение уравнения $\ddot{x} = a$ содержит две константы интегрирования ($x_0$ и $v_0$) — это и есть начальные условия задачи. Для уравнения $n$-го порядка нужно $n$ начальных условий.

**Обобщение на переменное ускорение.** Если $a_x(t)$ — произвольная функция времени, то:
$$v_x(t) = v_0 + \int\limits_0^t a_x(\tau)\,\mathrm{d}\tau, \qquad x(t) = x_0 + \int\limits_0^t v_x(\tau)\,\mathrm{d}\tau$$
Формулы 1, 2, 3 из таблицы выше — частный случай при $a_x = \text{const}$.

**Пример: линейно меняющееся ускорение.** Пусть $a_x(t) = \alpha t$ ($\alpha = \text{const}$). Тогда:
$$v_x(t) = v_0 + \int\limits_0^t \alpha\tau\,\mathrm{d}\tau = v_0 + \frac{\alpha t^2}{2}$$
$$x(t) = x_0 + v_0 t + \frac{\alpha t^3}{6}$$
Этот случай описывает, например, движение заряженной частицы в нарастающем электрическом поле (Kleppner, Example 1.11).
