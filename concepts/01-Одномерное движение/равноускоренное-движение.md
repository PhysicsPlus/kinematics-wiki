# Равноускоренное движение

Из бесконечного разнообразия движений с переменной скоростью есть одно особенно важное: движение с **постоянным ускорением** ($a_x = \text{const}$). Оно встречается повсеместно — свободное падение, разгон автомобиля с постоянным усилием, движение в постоянном электрическом поле. Для этого случая формулы кинематики выводятся аналитически и дают три компактных уравнения, охватывающих всю задачу. Строится на [[ускорение]] и [[мгновенная-скорость]].

## ☆ Скорость при постоянном ускорении

Если $a_x = \text{const}$, то ускорение — наклон графика $v_x(t)$. Постоянный наклон — прямая. Прямая, проходящая через точку $(0, v_0)$ с наклоном $a_x$:
$$v_x(t) = v_0 + a_x t$$
где $v_0 = v_x(0)$ — начальная скорость. Скорость меняется линейно по времени.

> [!example] Разгон мотоцикла
> Мотоцикл стартует из покоя ($v_0 = 0$) с ускорением $a_x = 3~\text{м/с}^2$. Найти скорость через $5~\text{с}$.
>
> $v_x(t) = 3t~\text{м/с}$. Через $t = 5~\text{с}$: $v_x = 15~\text{м/с} = 54~\text{км/ч}$.

## ☆ Координата при постоянном ускорении

Из определения скорости $v_x = \dot{x}$ и $v_x(t) = v_0 + a_x t$ следует:

Геометрически: площадь под графиком $v_x(t)$ на промежутке $[0, t]$ — трапеция с основаниями $v_0$ и $v_0 + a_x t$ и высотой $t$:
$$x(t) - x_0 = \frac{v_0 + (v_0 + a_x t)}{2}\cdot t = v_0 t + \frac{a_x t^2}{2}$$

Площадь под $v_x(t)$ — это всегда изменение координаты (в общем случае это интеграл, но здесь — трапеция). Итого:
$$x(t) = x_0 + v_0 t + \frac{a_x t^2}{2}$$

График $x(t)$ — парабола: ветви вверх при $a_x > 0$, вниз при $a_x < 0$.

> [!example] Мяч брошен вертикально вверх
> $v_0 = 20~\text{м/с}$, ось $y$ вверх, $a_y = -g = -10~\text{м/с}^2$. Найти максимальную высоту и время возвращения.
>
> $y(t) = 20t - 5t^2$, $v_y(t) = 20 - 10t$. Момент остановки: $t_* = 2~\text{с}$. Максимальная высота: $y(2) = 20~\text{м}$. Возвращение ($y = 0$): $t = 4~\text{с}$. Путь $L = 40~\text{м}$, перемещение $\Delta y = 0$.

## ☆ Формула без времени

Из двух уравнений $v_x = v_0 + a_x t$ и $x - x_0 = v_0 t + \frac{a_x t^2}{2}$ исключим $t$. Из первого: $t = \frac{v_x - v_0}{a_x}$. Подставляем во второе и раскрываем:
$$x - x_0 = \frac{v_0(v_x - v_0)}{a_x} + \frac{(v_x-v_0)^2}{2a_x} = \frac{(v_x-v_0)(v_x+v_0)}{2a_x} = \frac{v_x^2 - v_0^2}{2a_x}$$

Итого:
$$v_x^2 = v_0^2 + 2a_x(x - x_0)$$

Эта формула связывает скорость и перемещение напрямую, без времени. Удобна, когда время не нужно или неизвестно.

> [!definition]
> **Равноускоренное движение** ($a_x = \text{const}$). Три кинематических уравнения:
> $$v_x = v_0 + a_x t \qquad (v,\, a,\, t)$$
> $$x = x_0 + v_0 t + \frac{a_x t^2}{2} \qquad (x,\, v_0,\, a,\, t)$$
> $$v_x^2 = v_0^2 + 2a_x(x - x_0) \qquad (v,\, v_0,\, a,\, x)$$
> Из трёх уравнений выбирать то, которое содержит искомую и все данные.

## ☆ Свободное падение

Частный случай равноускоренного движения: ускорение $g \approx 9{,}8~\text{м/с}^2$ направлено вниз. Выбор знака зависит от направления оси:
- Ось $y$ вверх: $a_y = -g$
- Ось $y$ вниз: $a_y = +g$

В задачах удобно округлять $g \approx 10~\text{м/с}^2$, если не оговорено иное.

> [!example] Камень брошен вниз с крыши
> Крыша высотой $H = 45~\text{м}$, начальная скорость $v_0 = 5~\text{м/с}$ вниз (ось $y$ вниз). Когда камень ударится о землю?
>
> $$5t + 5t^2 = 45 \implies t^2 + t - 9 = 0 \implies t = \frac{-1 + \sqrt{37}}{2} \approx 2{,}54~\text{с}$$

## ★ Задачи с несколькими телами и сменой знака скорости

Из трёх уравнений таблицы использовать нужное — то, из которого можно найти искомую величину по данным. Всегда проверять: $v_x = 0$ — это мгновенная остановка, и тело может продолжить движение с обратным знаком.

> [!example] Тормозной путь автомобиля
> $v_0 = 108~\text{км/ч} = 30~\text{м/с}$, $a_x = -6~\text{м/с}^2$. Найти тормозной путь.
>
> Из формулы без времени: $0 = 900 - 12\Delta x \Rightarrow \Delta x = 75~\text{м}$ — около 15 длин автомобиля.

> [!example] Путь и перемещение при развороте
> $v_0 = 6~\text{м/с}$, $a_x = -2~\text{м/с}^2$. Найти путь и перемещение за $t = 5~\text{с}$.
>
> Момент остановки: $t_* = 3~\text{с}$ — внутри промежутка, тело разворачивается. Перемещение: $\Delta x = 6\cdot5 - \frac{2\cdot25}{2} = +5~\text{м}$. Путь до разворота: $L_1 = 9~\text{м}$; после разворота (за 2 с с нулевой начальной скоростью): $L_2 = 4~\text{м}$. Итого $L = 13~\text{м}$.

> [!example] Полицейский догоняет нарушителя
> Нарушитель едет с $v_1 = 30~\text{м/с}$. В $t = 0$ он проезжает мимо стоящего полицейского; тот стартует с $a = 5~\text{м/с}^2$. Когда и с какой скоростью полицейский догонит нарушителя?
>
> $x_{\text{нар}} = 30t$, $x_{\text{пол}} = \frac{5t^2}{2}$. Уравнение встречи: $\frac{5t^2}{2} = 30t \Rightarrow t = 12~\text{с}$. Скорость полицейского: $v = 60~\text{м/с}$ — вдвое больше скорости нарушителя.

## ★ Графики и геометрический смысл

График $v_x(t)$ — прямая линия с наклоном $a_x$ и начальным значением $v_0$. График $x(t)$ — парабола; вершина соответствует моменту остановки ($v_x = 0$).

Площадь под $v_x(t)$ равна изменению координаты. При $a_x = \text{const}$ это трапеция. Площадь со знаком: если $v_x < 0$ на части промежутка, соответствующая площадь отрицательна и вычитается из перемещения. Путь равен сумме модулей площадей на всех участках знакопостоянства $v_x$.

## ⭐ Строгий вывод через интегрирование уравнения движения

Исходное уравнение: $\ddot{x} = a = \text{const}$ — обыкновенное дифференциальное уравнение второго порядка.

Первое интегрирование по $t$ от $0$ до $t$:
$$\int\limits_0^t \frac{\mathrm{d}v_x}{\mathrm{d}\tau}\,\mathrm{d}\tau = \int\limits_0^t a\,\mathrm{d}\tau \implies v_x(t) - v_0 = at \implies v_x(t) = v_0 + at$$

Второе интегрирование:
$$\int\limits_0^t \frac{\mathrm{d}x}{\mathrm{d}\tau}\,\mathrm{d}\tau = \int\limits_0^t (v_0 + a\tau)\,\mathrm{d}\tau \implies x(t) - x_0 = v_0 t + \frac{at^2}{2}$$

Общее решение уравнения $\ddot{x} = a$ содержит две константы интегрирования ($x_0$ и $v_0$) — это начальные условия задачи. Для уравнения $n$-го порядка нужно $n$ начальных условий.

Обобщение на переменное ускорение: если $a_x(t)$ — произвольная функция времени, то:
$$v_x(t) = v_0 + \int\limits_0^t a_x(\tau)\,\mathrm{d}\tau, \qquad x(t) = x_0 + \int\limits_0^t v_x(\tau)\,\mathrm{d}\tau$$

Формулы таблицы выше — частный случай при $a_x = \text{const}$. Пример с линейным ускорением: $a_x(t) = \alpha t$:
$$v_x(t) = v_0 + \frac{\alpha t^2}{2}, \qquad x(t) = x_0 + v_0 t + \frac{\alpha t^3}{6}$$
Этот случай описывает движение заряженной частицы в нарастающем электрическом поле (Kleppner, Example 1.11).
