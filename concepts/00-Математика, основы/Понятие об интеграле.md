# Интеграл

Интегрирование — операция, обратная дифференцированию. Она отвечает на вопрос: если известен темп изменения величины, как найти саму величину? В кинематике: знаем скорость $v(t)$ — найти перемещение $x(t)$. Эта заметка содержит лишь начальные сведения; тема весьма обширна и требует отдельного развёрнутого изучения. Предполагается знакомство с [[Производная|производной]].

## ☆ Площадь под графиком

Дан произвольный график скорости $v(t)$. Как найти перемещение за промежуток от $t_1$ до $t_2$? При постоянной скорости ответ прост: $\Delta x = v\cdot(t_2-t_1)$ — площадь прямоугольника под графиком. При переменной скорости идея та же, но нужно аккуратнее.

Разобьём промежуток $[t_1, t_2]$ на $N$ малых шагов длиной $\Delta t = \frac{t_2 - t_1}{N}$. На каждом шаге будем считать скорость приближённо постоянной и равной $v_i = v(t_i)$. Перемещение за $i$-й шаг:
$$\Delta x_i \approx v_i \cdot \Delta t$$
Суммируем:
$$\Delta x \approx \sum_{i=1}^{N} v_i \cdot \Delta t$$
Каждое слагаемое $v_i\cdot\Delta t$ — площадь прямоугольника шириной $\Delta t$ и высотой $v_i$. Сумма этих прямоугольников приближает площадь фигуры под графиком $v(t)$.

При $N \to \infty$ (шаги становятся всё мельче) прямоугольники всё точнее заполняют фигуру, и приближение становится точным. Вывод:

перемещение = площадь под графиком $v(t)$.

> [!example] Равномерное движение
> Тело движется со скоростью $v = 5$ м/с от $t_1 = 0$ до $t_2 = 4$ с. График — горизонтальная прямая. Найти перемещение.
>
> Площадь прямоугольника:
> $$\Delta x = 5 \cdot 4 = 20\text{ м}$$
> Сумма прямоугольников даёт ровно $20$ м при любом числе шагов — потому что фигура и сама является прямоугольником.

> [!example] Равноускоренное движение
> Тело стартует из покоя и разгоняется с ускорением $a = 2$ м/с²; скорость $v(t) = 2t$. Найти перемещение за $[0, 3]$ с.
>
> Площадь треугольника:
> $$\Delta x = \frac{1}{2}\cdot v(3)\cdot 3 = \frac{1}{2}\cdot 6\cdot 3 = 9\text{ м}$$
> Проверим суммой прямоугольников при $N = 3$ (шаги $\Delta t = 1$ с):
> $$\sum v_i\cdot\Delta t = v(0)\cdot 1 + v(1)\cdot 1 + v(2)\cdot 1 = 0 + 2 + 4 = 6\text{ м}$$
> Ошибка $33\%$ — прямоугольники срезают углы. При $N = 6$ (шаги $\Delta t = 0{,}5$ с) ошибка уменьшается до $17\%$; при $N \to \infty$ сумма стремится к точному значению $9$ м.

Знак перемещения. Если скорость отрицательна ($v < 0$), то тело движется в обратном направлении, и соответствующие слагаемые $v_i\cdot\Delta t < 0$. Площадь ниже оси вычитается. Итоговое перемещение — алгебраическая сумма площадей выше и ниже оси.

Тот же принцип работает для ускорения: изменение скорости = площадь под графиком $a(t)$.

## ★ Первообразная и определённый интеграл

Эти два понятия тесно связаны, но концептуально различны.

> [!definition]
> **Первообразная.** Функция $F(x)$ называется первообразной функции $f(x)$ на некотором промежутке, если $F'(x) = f(x)$ для всех $x$ на этом промежутке. Полное семейство первообразных: $F(x) + C$, где $C$ — произвольная константа.

Если $F$ — одна первообразная, то $F + C$ (где $C$ — любая константа) тоже первообразная, поскольку $(F+C)' = F' = f$. Других первообразных нет: если $F'(x) = G'(x) = f(x)$, то $(F-G)' = 0$, а значит $F - G = \mathrm{const}$.

> [!definition]
> **Неопределённый интеграл** — обозначение для семейства всех первообразных:
> $$\int f(x)\,\mathrm{d}x = F(x) + C$$
> Константа $C$ называется константой интегрирования — она определяется из начальных или граничных условий задачи.

> [!definition]
> **Определённый интеграл** определяется через предел суммы Римана:
> $$\int\limits_a^b f(x)\,\mathrm{d}x = \lim_{N\to\infty}\sum_{i=1}^{N} f(x_i)\,\Delta x$$
> где $\Delta x = \frac{b-a}{N}$, а $x_i$ — точка в $i$-м шаге. Результат — число, равное площади под графиком $f$ (с учётом знака).

Символ $\int$ — вытянутая буква S (от *summa*); пределы $a$ и $b$ — нижний и верхний.

> [!definition]
> **Теорема Ньютона–Лейбница.** Если $F$ — первообразная $f$, то:
> $$\int\limits_a^b f(x)\,\mathrm{d}x = F(b) - F(a)$$

Эта теорема позволяет вычислять определённые интегралы без вычисления предела суммы — достаточно найти первообразную. Это главный практический инструмент.

> [!example] Перемещение при равноускоренном движении
> Тело движется по закону $v(t) = 2t$ м/с. Найти перемещение за $[0, 3]$ с.
>
> Первообразная: $F(t) = t^2$ (поскольку $(t^2)' = 2t$). По теореме Ньютона–Лейбница:
> $$\int\limits_0^3 2t\,\mathrm{d}t = F(3) - F(0) = 9 - 0 = 9\text{ м}$$
> Совпадает с геометрическим ответом — площадь треугольника $\frac{1}{2}\cdot 6\cdot 3 = 9$ м.

## ★ Интегрирование стандартных выражений

Таблица интегралов — это таблица производных, прочитанная наоборот: если $(F)' = f$, то $\int f\,\mathrm{d}x = F + C$.

| $f(x)$ | $\int f(x)\,\mathrm{d}x$ | Условие |
|:------:|:------------------------:|:-------:|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ | $n \neq -1$ |
| $\frac{1}{x}$ | $\ln|x| + C$ | $x \neq 0$ |
| $e^x$ | $e^x + C$ | |
| $a^x$ | $\frac{a^x}{\ln a} + C$ | $a > 0,\; a \neq 1$ |
| $\sin x$ | $-\cos x + C$ | |
| $\cos x$ | $\sin x + C$ | |
| $\frac{1}{\cos^2 x}$ | $\operatorname{tg} x + C$ | |
| $\frac{1}{\sqrt{1-x^2}}$ | $\arcsin x + C$ | |
| $\frac{1}{1+x^2}$ | $\operatorname{arctg} x + C$ | |

Линейность интегрирования:
$$\int\bigl(f(x) + g(x)\bigr)\,\mathrm{d}x = \int f(x)\,\mathrm{d}x + \int g(x)\,\mathrm{d}x, \qquad \int C\,f(x)\,\mathrm{d}x = C\int f(x)\,\mathrm{d}x$$

Интегрирование по частям. Из правила Лейбница $(uv)' = u'v + uv'$ следует $uv' = (uv)' - u'v$. Проинтегрируем обе части:
$$\int u\,\mathrm{d}v = uv - \int v\,\mathrm{d}u$$
где для краткости $\mathrm{d}u = u'\,\mathrm{d}x$ и $\mathrm{d}v = v'\,\mathrm{d}x$. Метод применяется, когда подынтегральное выражение является произведением двух функций, и одну из них удобно дифференцировать, другую — интегрировать.

> [!example] Интегрирование по частям: $\int x e^x\,\mathrm{d}x$
> Найти $\int x e^x\,\mathrm{d}x$.
>
> Положим $u = x$, $\mathrm{d}v = e^x\,\mathrm{d}x$. Тогда $\mathrm{d}u = \mathrm{d}x$ и $v = e^x$:
> $$\int x e^x\,\mathrm{d}x = x e^x - \int e^x\,\mathrm{d}x = x e^x - e^x + C = e^x(x-1) + C$$
> Проверка: $[e^x(x-1)]' = e^x(x-1) + e^x\cdot 1 = xe^x$ ✓.

> [!example] Интегрирование по частям: $\int x\sin x\,\mathrm{d}x$
> Найти $\int x\sin x\,\mathrm{d}x$.
>
> Положим $u = x$, $\mathrm{d}v = \sin x\,\mathrm{d}x$. Тогда $\mathrm{d}u = \mathrm{d}x$ и $v = -\cos x$:
> $$\int x\sin x\,\mathrm{d}x = -x\cos x - \int(-\cos x)\,\mathrm{d}x = -x\cos x + \sin x + C$$

Заметка содержит лишь начальные сведения об интегрировании. За пределами этой заметки остаются: метод подстановки (замена переменной), интегрирование рациональных дробей (разложение на простые дроби), тригонометрические подстановки, и многое другое. Интегрирование в полном объёме рассматривается в разделе [[concepts/07-Математика, продвинутая/Дифференциальные уравнения/index|ОДУ]] и в отдельных тематических заметках.
