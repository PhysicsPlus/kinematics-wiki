# Математика: основы

Всё необходимое для работы с кинематикой — с нуля. Предполагается только знание квадратного уравнения и элементарной алгебры.

## Заметки

### Анализ: цепочка малые величины → предел → производная → ряд Тейлора

| Заметка | Уровень | Статус | Ключевая идея |
|---------|---------|--------|---------------|
| [[малые-величины]] | 1–2 | ✅ | Приближения, геометрия малых углов, оценка точности, распространение погрешностей, базовые ряды |
| [[предел]] | 1–3 | ✅ | Определение Коши; $\lim\frac{\sin x}{x} = 1$; бесконечно малые; непрерывность |
| [[производная]] | 2–3 ⭐ | ✅ | Точный коэффициент линеаризации $C$; геом. смысл; правила; $\frac{\mathrm{d}\hat{\mathbf{e}}}{\mathrm{d}t} = \boldsymbol{\omega}\times\hat{\mathbf{e}}$ |
| [[Ряд Тейлора]] | 3 ⭐ | ✅ | Коэффициенты ряда через производные; остаточный член; таблица разложений |
| [[понятие-об-интеграле]] | 2–3 ⭐ | ⚠️ черновик | Площадь под графиком; формула Ньютона–Лейбница |
