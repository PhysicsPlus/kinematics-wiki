# Полярные координаты

Описание движения на плоскости в координатах $(r, \varphi)$. Естественный язык для задач с центральной симметрией.

## Планируемые заметки

- [[полярные-основы]] — определение, связь с декартовыми, орты $\hat{\mathbf{e}}_r$, $\hat{\mathbf{e}}_\varphi$
- [[скорость-в-полярных]] — радиальная и трансверсальная компоненты
- [[ускорение-в-полярных]] — вывод, центростремительная и кориолисова компоненты
- [[угловая-скорость-радиуса]] — связь $\dot\varphi$ с поперечной скоростью
