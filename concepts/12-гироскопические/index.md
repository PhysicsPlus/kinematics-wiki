# Движение при наличии гироскопических сил

Силы вида $\vec{F} \perp \vec{v}$: не совершают работы, не меняют модуль скорости, только поворачивают её.

## Планируемые заметки

- [[гироскопическая-сила-общее]] — свойство $\vec{F} \perp \vec{v}$: $|\vec{v}| = \text{const}$, работа равна нулю
- [[движение-в-магнитном-поле]] — сила Лоренца, циклотрон, ларморовский радиус
- [[скрещенные-поля]] — $\vec{E} \perp \vec{B}$: дрейф, скорость дрейфа
- [[кориолис-как-гироскопическая]] — связь с гл. 13, отклонение тел на Земле
