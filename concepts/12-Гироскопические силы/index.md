# Движение при наличии гироскопических сил

Силы вида $\mathbf{F} \perp \mathbf{v}$: не совершают работы, не меняют модуль скорости, только поворачивают её.

## Планируемые заметки

- [[гироскопическая-сила-общее]] — свойство $\mathbf{F} \perp \mathbf{v}$: $|\mathbf{v}| = \text{const}$, работа равна нулю
- [[движение-в-магнитном-поле]] — сила Лоренца, циклотрон, ларморовский радиус
- [[скрещенные-поля]] — $\mathbf{E} \perp \mathbf{B}$: дрейф, скорость дрейфа
- [[кориолис-как-гироскопическая]] — связь с гл. 13, отклонение тел на Земле
