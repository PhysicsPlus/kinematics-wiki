# Цилиндрические координаты

Координаты $(r, \varphi, z)$: естественный язык для задач с осевой симметрией и пространственного вращения.

## Планируемые заметки

- [[цилиндрические-основы]] — связь с декартовыми, орты $\hat{\mathbf{e}}_r$, $\hat{\mathbf{e}}_\varphi$, $\hat{\mathbf{e}}_z$
- [[скорость-в-цилиндрических]] — три компоненты, вывод через вращающийся базис
- [[ускорение-в-цилиндрических]] — центростремительное, кориолисово и осевое слагаемые
- [[винтовое-движение]] — равномерное движение по спирали: шаг, угол подъёма

## Задачи

- Шнек (движение по резьбе)
- Точка на вращающемся конусе
- Пространственный маятник
