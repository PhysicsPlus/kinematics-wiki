- Скалярное произведение: $\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}| |\mathbf{b}| \cos\theta = a_x b_x + a_y b_y + a_z b_z$
- Векторное произведение: $\mathbf{a} \times \mathbf{b} = (a_y b_z - a_z b_y, \ldots)$, $|\mathbf{a} \times \mathbf{b}| = |\mathbf{a}| |\mathbf{b}| \sin\theta$

### Дифференцирование
- Правило цепи: $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$
- Полный дифференциал: $df = \frac{\partial f}{\partial x} dx + \frac{\partial f}{\partial y} dy + \frac{\partial f}{\partial z} dz$
- Градиент: $\nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$

### Интегрирование
- Основная теорема анализа: $\int_a^b f'(x) dx = f(b) - f(a)$
- Интегрирование по частям: $\int u \, dv = uv - \int v \, du$
- Замена переменной: $\int_a^b f(x) dx = \int_{g(a)}^{g(b)} f(g(u)) g'(u) du$

### ОДУ 1-го порядка
- Уравнение с разделяющимися переменными: $\frac{dy}{dx} = f(x) g(y)$ → $\int \frac{dy}{g(y)} = \int f(x) dx + C$
- Линейное: $\frac{dy}{dx} + P(x)y = Q(x)$ → $y = e^{-\int P dx}\left(C + \int Q e^{\int P dx} dx\right)$

### ОДУ 2-го порядка
- Линейное однородное с постоянными коэффициентами: $y'' + py' + qy = 0$

- Разложения: $e^x = \sum_{n=0}^\infty \frac{x^n}{n!}$, $\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!}$, $\cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!}$
