# Промпт для Qwen: авторские разработки по приближённым вычислениям

Скопируй этот промпт в Qwen вместе с обоими LaTeX-файлами.

---

## Промпт

Ты анализируешь **два авторских учебных материала** по приближённым вычислениям в физике и математике (файлы в формате LaTeX).

Это не учебник, а авторские разработки для физ-мат курса по кинематике. Твоя задача — составить подробную сводку содержания обоих файлов для базы знаний. Пиши на **русском языке**.

**Для каждого файла выдай:**

```markdown
## Файл: [имя файла или условное название]

### Общая тема и назначение
[Что это — конспект, листок задач, теоретическая разработка? Для какой аудитории?]

### Структура
[Разделы и подразделы с кратким описанием]

### Основные концепции и методы
[Перечень ключевых идей, приёмов, алгоритмов]

### Ключевые формулы и приближения
[Все формулы в LaTeX — $...$ для строчных, $$...$$ для выключных]

### Примеры и задачи
[Список задач/примеров с кратким описанием физического смысла]

### Связь с другими темами курса
[К чему это относится: малые величины, ряд Тейлора, оценки порядка величины, метод малого параметра, ...]

### Что стоит вынести в отдельную заметку базы знаний
[Конкретные темы, которые заслуживают отдельной заметки в базе]
```

**После разбора обоих файлов добавь:**

```markdown
## Сравнение и связи между файлами
[Как два материала соотносятся друг с другом: дополняют, пересекаются, разный уровень?]

## Рекомендуемая структура заметок для базы знаний
[Предложи конкретный список заметок в формате:]
- `имя-заметки` (уровень 1/2/3) — краткое описание
- `имя-заметки` ⭐ (уровень 3) — краткое описание
```

**Контекст базы знаний:**
Уже есть заметка [[concepts/00а-математика/малые-величины]] с приближениями:
$\sin x \approx x$, $\cos x \approx 1$, $(1+x)^n \approx 1+nx$ при малом $x$.

Авторские разработки должны её расширить. Интересует всё, что касается:
- Оценки точности приближений
- Ряд Тейлора и его применение в физике
- Метод малого параметра
- Оценки порядка величины (задачи типа Ферми)
- Геометрические приближения
- Применения в кинематике конкретно

**Формат LaTeX в базе знаний:** формулы в `$...$` и `$$...$$`, пакеты physics и unicode-math.
