## Промпт

Ты анализируешь главы из учебника **Young & Freedman "University Physics with Modern Physics"**.

Твоя задача — составить подробное саммари каждой главы для базы знаний по кинематике. Пиши на **русском языке**.

**Для каждой главы выдай:**

```markdown
## Chapter X. [Название главы]

### Основные концепции
[Перечень ключевых понятий, введённых в главе]

### Содержание по разделам
- §X.Y [название раздела], с. XX–XX: [что рассматривается, какие формулы выводятся]
- §X.Y ...

### Ключевые формулы
[Самые важные формулы главы — в LaTeX]

### Примеры и задачи
- с. XX: [краткое описание типичного примера]
- с. XX: [краткое описание]

### Соответствие главам базы знаний
[Укажи, каким главам нашей базы соответствует материал:]
- 01 (одномерная кинематика)
- 02 (относительное движение)
- 03 (криволинейное движение)
- 04а (плоскопараллельное движение АТТ)
- 05 (полярные координаты)
- 06 (баллистика)
- и т.д.
```

**Требования:**
- Все формулы в LaTeX: `$...$` для строчных, `$$...$$` для выключных
- Номера страниц указывай точно — они нужны для ссылок
- Не пропускай примеры и иллюстративные задачи — они важны для базы
- Если тема выходит за рамки кинематики (динамика, энергия) — отметь, но кратко

**Главы базы знаний для справки:**
- 00а: математика (векторы, тригонометрия, малые величины)
- 00б: ОДУ
- 01: одномерная кинематика (скорость, ускорение, равноускоренное движение)
- 02: относительное движение, системы отсчёта
- 03: криволинейное движение (окружность, нормальное/касательное ускорение, радиус кривизны)
- 04а: плоскопараллельное движение АТТ (МЦС, поле скоростей)
- 04б: АТТ в 3D (углы Эйлера, винтовое движение)
- 05: полярные координаты
- 06: баллистика (+ огибающие)
- 07: задачи преследования
- 08: геометрия кривых, формулы Френе–Серре
- 09: цилиндрические координаты
- 10: движение в центральном поле
- 11: кинематика механизмов
- 12: гироскопические силы
- 13: общий случай преобразования СО
- 14: оптико-механическая аналогия
