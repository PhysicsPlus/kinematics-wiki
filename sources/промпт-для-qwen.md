# Промпт для Qwen: создание сводки учебника

Используй этот промпт при загрузке PDF учебника в Qwen.
Результат скопируй в файл `sources/ИМЯ-ПАПКИ/index.md`.

---

## Промпт

Ты анализируешь учебник по механике/теоретической механике.
Твоя задача — создать структурированную сводку для базы знаний по кинематике.

**Формат ответа — строго Markdown, начиная с фронтматтера:**

```markdown
---
type: source
title: [полное название]
author: [авторы]
year: [год издания]
file: [имя файла.pdf]
bib_key: [если знаешь ключ из BibTeX]
level: [1 = школьник / 2 = старшеклассник физ-мат / 3 = студент теормех]
language: [russian / english]
pages_total: [число страниц]
---

# [Автор] — [Название]

## Аннотация
[2–3 предложения: что это за книга, её место в литературе, особенности подачи]

## Кинематика — содержание по страницам

[Для каждого раздела/главы, относящегося к кинематике:]

### [Название раздела/главы]
- с. X–Y: [краткое описание темы]
- с. X–Y: [краткое описание темы]
...

## Задачи по кинематике
[Если есть задачи:]
- с. X–Y: задачи на [тему], уровень [сложности]

## Особенности
[Нотация, отличия от стандартной, на что обратить внимание]
```

**Требования:**
- Включай ТОЛЬКО разделы, относящиеся к кинематике
- Для каждого диапазона страниц — конкретная тема, не просто «§3»
- Если тема соответствует одной из глав базы знаний (список ниже) — укажи в скобках

**Главы базы знаний:**
- 00а: математика (векторы, тригонометрия)
- 00б: ОДУ
- 00в: дифгеометрия кривых
- 01: одномерная кинематика
- 02: относительное движение, системы отсчёта
- 03: криволинейное движение (окружность, нормальное/касательное ускорение, радиус кривизны)
- 04а: плоскопараллельное движение АТТ (МЦС, поле скоростей)
- 04-Плоско-параллельное движение: АТТ в 3D (углы Эйлера, винтовое движение)
- 05: полярные координаты
- 06: баллистика
- 07: задачи преследования
- 08: геометрия кривых, формулы Френе–Серре
- 09: цилиндрические координаты
- 10: движение в центральном поле (Бине, Рунге–Ленц, годограф)
- 11: кинематика механизмов
- 12: гироскопические силы
- 13: общий случай преобразования СО (теорема Кориолиса)
- 14: оптико-механическая аналогия
