## Как добавить новый источник

### PDF-книга (учебник, монография)

1. **Создать папку** `sources/имя-книги/` (транслит или английское имя, через дефис)
2. **Положить PDF** в папку (целиком или отдельные главы). PDF не попадает в git (`.gitignore`).
3. **Написать промпт для Qwen** — сохранить как `sources/промпт-имя-книги.md`. Образец: `промпт-young-freedman.md`. Промпт содержит:
   - Формат вывода (markdown с LaTeX-формулами)
   - Требование указывать номера страниц
   - Список глав базы для маппинга (00а–14)
   - Специфику книги (уровень, язык, какие разделы особенно важны)
4. **Загрузить в Qwen** нужные главы PDF + промпт, получить сводку
5. **Сохранить результат** как `sources/имя-книги/index.md` (или отдельные файлы по главам)
6. **Добавить строку** в таблицу выше (источник, уровень, статус)

### LaTeX-исходник (авторский материал)

1. **Положить `.tex` файл** в `sources/chapters/`
2. **Добавить строку** в таблицу «LaTeX-исходники» выше
3. Файлы `.tex` **коммитятся в git** (текстовые, не бинарные)

### Статья, задачник, отдельный материал

1. **Создать папку** или положить в существующую
2. **При необходимости** — написать промпт для Qwen
3. **Добавить в реестр** выше

## Правила именования папок

- Латиница или кириллица, строчные буквы, через дефис: `young-freedman`, `мякишев-10`, `яковлев-математика`
- Промпт для Qwen: `промпт-[то же имя].md`
- Сводка: `[папка]/index.md`
