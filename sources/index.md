## Как добавить новый источник

1. Создай папку `sources/ИМЯ/`
2. Загрузи PDF в неё (он не попадёт в git — внесён в .gitignore)
3. Дай PDF в Qwen с [[промпт-для-qwen|этим промптом]]
4. Сохрани результат как `sources/ИМЯ/index.md`
5. Добавь строку в таблицу выше
