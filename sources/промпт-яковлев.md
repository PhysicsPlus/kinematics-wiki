# Промпт для Qwen: Яковлев — Математика для физики (mathus.ru)

Скопируй этот промпт в Qwen вместе с обоими PDF-файлами.

---

## Промпт

Ты анализируешь **два учебных материала** репетитора Яковлева И.В. с сайта mathus.ru — «Математика для физики».

Это качественные методические разработки уровня физ-мат школы и младших курсов вуза. Твоя задача — составить подробную сводку содержания обоих файлов для базы знаний по кинематике. Пиши на **русском языке**.

**Для каждого файла выдай:**

```markdown
## Файл: [название или условное обозначение]

### Тема и назначение
[Что именно разбирается, для какой аудитории]

### Структура
[Разделы с кратким описанием]

### Основные концепции и методы
[Ключевые идеи, приёмы, алгоритмы]

### Ключевые формулы
[Все формулы в LaTeX]

### Примеры и задачи
[Характерные задачи с кратким описанием]

### Связь с физикой/кинематикой
[Конкретные применения в механике и кинематике]

### Рекомендуемые заметки для базы знаний
[Что стоит вынести в отдельные заметки:]
- `имя-заметки` (уровень 1/2/3⭐) — описание
```

**После разбора обоих файлов:**

```markdown
## Связи между файлами
[Как соотносятся: дополняют, пересекаются, разный уровень?]

## Итоговая структура заметок для базы
[Полный список заметок, которые стоит создать]
```

**Контекст базы знаний — математические разделы:**
- `00а`: предел, производная⭐, интеграл⭐, векторы, тригонометрия, малые величины, поворот координат
- `00б`: ОДУ (разделение переменных, линейные уравнения 1-го и 2-го порядка, осциллятор)
- `00в`: параметрические кривые, кривизна, кручение, эволюта/эвольвента
- `приближения-авт`: авторские разработки по приближённым вычислениям (уже есть)

Особый интерес представляет материал по:
- Приближениям и оценкам (дополнит авторские разработки)
- Тригонометрии для физических задач
- Работе с малыми величинами и рядами
- Геометрии для механики

**Формат формул:** `$...$` и `$$...$$`, пакеты physics и unicode-math.
